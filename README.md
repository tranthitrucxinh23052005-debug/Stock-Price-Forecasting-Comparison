# 📈 Comparative Analysis of Stock Price Prediction Models

## 📌 Abstract
This research explores the effectiveness of three distinct modeling approaches—Statistical (ARIMA), Machine Learning (Random Forest), and Deep Learning (LSTM)—in forecasting S&P 500 stock prices. The study evaluates performance across 172 stock tickers to identify the most robust predictive framework for financial time-series data.

## 🛠 Methodology & Features
* **Data Source:** S&P 500 Stocks dataset via KaggleHub.
* **Technical Indicators:** Integrated RSI, MACD, and Bollinger Bands to enhance feature depth.
* **Model Benchmarking:**
    * **ARIMA:** Baseline for linear time-series trends.
    * **Random Forest:** Capturing non-linear interactions between technical indicators.
    * **LSTM:** Utilizing Recurrent Neural Networks (RNN) for long-term temporal dependencies.

## 📊 Performance Summary
| Model | Strength | Best Use Case |
| :--- | :--- | :--- |
| **ARIMA** | Simplicity & Interpretability | Stable, linear market trends |
| **Random Forest** | Feature Importance analysis | High-dimensional data with technical indicators |
| **LSTM** | High predictive accuracy | Volatile markets with complex temporal patterns |
